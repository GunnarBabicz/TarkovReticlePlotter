This is a testing environment for the standalone Reticle Painter for Escape from Tarkov

To activate the Python Environment, open a powershell window in this path. Perform the following steps to instantiate this environment: 

1. Navigate to the path the folder in contained in in Powershell
2. Activate the virtual environment with .\.venv\Scripts\Activate.ps1
3. Install the needed packages with the command pip install -r requirements.txt
4. To create a sample output, enter python run_in_command_line.py
